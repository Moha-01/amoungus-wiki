Colors

Colors change the appearance of the characters in Among Us. Only one character can have each color in the same lobby.
  • Overview
  • List of Colors
  • Trivia

Overview

Players are able to select between 18 different colors using the customize ability in the lobby: Red, Blue, Green, Pink, Orange, Yellow, Black, White, Purple, Brown, Cyan, Lime, Maroon, Rose, Banana, Gray, Tan and Coral. The player's selected color does not directly affect gameplay, and only directly affects the player's in-game appearance. The only exception was Fortegreen, who was not legitimately playable in-game and caused numerous bugs when played as. Each color also has a unique color ID; for example, Red's is 0, and Yellow's is 5. Fortegreen was an exception to this rule, however, as it occupied all IDs of whole numbers above 11, examples being 12, 13, and 100. As of version 2021.2.21, Fortegreen's color ID was removed from the code so it cannot be accessed by ID.

Trivia

  • There are eight colors that were added after Among Us's initial release.
    • Cyan and Lime were added in version 2019.4.24.[1]
    • Six new colors were confirmed for a future Among Us update in the Innersloth Discord server.[2] This update was implemented to the game on June 15, 2021.
      • On Discord, a tiebreaker vote between Coral and Olive started on April 20, 2021 and ended roughly 28 hours later.[3] After the vote ended, Coral was deemed the winner.
      • On May 20, 2021, Innersloth confirmed another new color, Rose, on the Discord server and Twitter. They also announced more information to be announced during the Summer Game Fest on June 10.[4]
      • On May 26, 2021, Gray was confirmed to be added by Innersloth on the Discord server and Twitter.
      • On June 3, 2021, Tan's readdition was confirmed by Innersloth on Twitter and Discord, with a trailer.[6]
      • On June 8 2021, a couple days later, Maroon was confirmed as an upcoming color by Innersloth on their official Discord server and Twitter account.[7]
      • On June 10, 2021, during the Summer Game Fest, a color called Banana was confirmed.[8]
  • Fortegreen and Olive are the only colors that cannot be selected via customize
    • However, these color IDs result from Fortegreen being Among Us's failsafe color.
  • Tan is the only color that was temporarily removed from Among Us. During the removal period, it was still officially recognized in the Innersloth Discord server as a custom emoji, and it has since been announced that it will be added back.
  • If the player's selected color ID is anything but a whole number, it will be set to a random value between 0 and 17.
  • Some cosmetics change their appearance based on the player's selected color. These cosmetics include the Crewmate hat, balloon, and Mini Crewmate.
  • If the player is playing as Fortegreen, then these cosmetics will take the appearance of Red's.
  • Contrary to popular belief, the player's color does not affect the chances of them being The Impostor.
